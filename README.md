## About This Project

This repository aims to help beginners **quickly master the Finite Element Method (FEM)** by implementing the entire process from shape function construction, element stiffness matrix assembly to 1D/2D problem solving through **MATLAB code**.  
All codes are accompanied by **detailed comments** and **formula derivation explanations**, allowing you to quickly understand the mathematical logic behind FEM (｡•̀ᴗ-)✧

**Target Audience**:  
- Undergraduate/graduate students studying *Finite Element Method*  
- Engineers who want to write FEM code from scratch in MATLAB  
- Anyone interested in numerical computing and numerical solutions of partial differential equations

## Quick Start

### Environment Requirements
- MATLAB R2020b or later (older versions may also work, probably?)  
- MATLAB Basic Toolboxes (Required: MATLAB Symbolic Math Toolbox)

### Run Examples
All examples are provided as MATLAB Live Scripts (`.mlx`) with step-by-step formula explanations and built-in visualizations. After cloning or downloading this repository, navigate to the repo folder in MATLAB, then run the scripts directly:
```matlab
% Navigate to your local repository path (replace with your actual folder path)
>> cd path/to/your/MATLAB-finite-element-calculation

% Run the example scripts in order
>> run example1.mlx   % 
>> run example2.mlx   % 
>> run example3.mlx   % 
>> run example4.mlx   %
